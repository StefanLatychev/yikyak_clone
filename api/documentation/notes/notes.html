GET /api/notes.php Get Notes

Get a number of notes from the database, either by location or globally. If time is given, it must be in UTC format.

URL

Information

Authentication required?Y
Request formatjson
Response formatjson

Parameters

JSON


{
	"location": {
		"latitude": (float)
		"longitude": (float)
	},
	"time": {
		"timestamp": "",
		"direction": ""
	},
	"max_notes": (int),
	
}
ParameterRequired?Description
locationNIf provided, will return notes from the given location (latitude, longitude). If not provided, with get notes worldwide
latitude-Geographical latitude (must also provide longitude).
longitude-Geographical longitude (must also provide latitude).
timeNIf provided, will get notes from the given timestamp either forwards, or backwards in time (determined by direction).
timestamp-Y-m-d H:i:s timestamp in UTC
direction-Either after or before, coresponding to after or before the given timestamp.
max_notesNIf provided, will only return max_notes worth of notes. Otherwise will return the default amount.

Response

JSON


{
	"errors": [],
	"status": "",
	"notes": [
		{
			"id": "",
			"time": "",
			"location_latitude": "",
			"location_longitude": "",
			"votes": "",
			"message": ""
		}
	]
}
ParameterDescription
errorsArray of errors that occured when processing the request
statusHTTP status code of request
notesArray of notes that fit the request's criteria sorted in ascending time order starting at lowest array index
idinternal id of note
timeUTC time the note was posted
location_latitudelatitude where the note was posted
location_longitudelongitude where the note was posted
votesvotes associated with post
messagecontent of note

If status has a 200 status code, the request succeeded.
