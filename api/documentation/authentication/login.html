POST /api/authentication.php Login

Used to authenticate user and returns an API session key. Sets cookie containing session key on user's machine. Cookie is set to only be sent by browser over https connection.

If the request contains valid credentials and there are existing active API session keys for that user, the active keys will be invalidated.
