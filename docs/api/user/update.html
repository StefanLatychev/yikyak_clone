PUT /api/user.php Update User Information

Updates user information. Only updates the information provided.

  • Emails must be of correct format with alpha-numeric characters, and up to 50 characters in length.
  • Phone number must only contain digits, and between 7 and 15 characters in length.
  • Passwords must be between 8 and 100 characters in length and accepts all alpha-numeric values and the symbols ~!@#$%^&*().
